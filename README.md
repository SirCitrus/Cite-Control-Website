# Cite Control Website
Static HTML pages for the Cite Control landing, pricing, support, privacy policy, and terms of service.

## Pages
- **index.html** — Homepage with hero section, user types, beta signup, features, and testimonials
- **pricing.html** — Pricing tiers (Community, Premium, Agency) + Lookout booking rates
- **support.html** — FAQ with 8 sections covering all features and user types
- **privacy.html** — Privacy policy and data handling practices
- **terms.html** — Terms of service and usage agreements
